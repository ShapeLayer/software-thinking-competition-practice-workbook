# Practice Workbooks of Software Thinking Competition

_SW사고력올림피아드 대회 준비 워크북_

<br />

SW사고력올림피아드(現 AI・SW사고력올림피아드) 대회 준비를 위해 작성한 워크북입니다.  

《문해・사고・서술》은 글을 읽고, 글이 의도하는 바를 구체적으로 판단, 이렇게 확인한 바에 대해 자신의 생각을 정리하고, 이 생각을 효과적이고 논리적으로 글로 작성해내는 능력을 기르는 것을 목표로 합니다. 입력(읽기), 처리(문해, 사고, 정리), 출력(글쓰기) 단계를 빠르고 정확하게 수행하는 훈련을 통해 대회에서 요구하는 사고력과 표현력을 키울 수 있도록 구성되어 있습니다.  

《소프트웨어・데이터》는 소프트웨어 설계와 데이터 분석에 필요한 기본적인 개념과 도구들을 익히는 것을 목표로 합니다. 흐름도, 집합 등을 활용해 상황을 이해, 객체 간의 관계를 파악, 문제 해결을 위한 논리적 사고를 기르는 훈련을 통해 대회에서 요구하는 소프트웨어적 사고력과 데이터 분석 능력을 키울 수 있도록 구성되어 있습니다.  

<br />

하지만 이 워크북은 대회의 기출 문제를 직접적으로 다루거나, 대회에서 사용하는 문제 유형을 그대로 재현하지는 않습니다. 이 교재는 대회에서 마주할 수 있는 문제 유형과 서술 요구 사항에 충분히 대응 가능하도록 준비되었지만, 실제로 대회에서 사용된 문제를 찾고자 한다면 다른 자료를 참고하는 것이 좋습니다.
